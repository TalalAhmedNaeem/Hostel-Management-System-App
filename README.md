Hostel Management System GUI App

A comprehensive Windows-based GUI application designed to streamline hostel administration.

Project Overview:

The Hostel Management System App is a robust solution for managing student data, tracking fee payments, and monitoring living status. It provides an intuitive interface for administrators to efficiently manage hostel operations.

Key Features:

- Student Management: Register and manage student profiles, including personal details and room assignments.
- Fee Payment Tracking: Record and track fee payments, ensuring timely updates and accurate records.
- Living Status Updates: Instantly update and monitor student living status, enabling proactive interventions.
- Data Grid View: Display student data in a tabular format for easy browsing and management.

How to Run:

1. Clone the repository from GitHub.
2. Open the project in Visual Studio.
3. Restore NuGet packages and build the solution.
4. Update the SQL Server connection string in the Function.cs file.
5. Run the application using Visual Studio or the executable file.

Tools and Technologies:

- Windows Forms: GUI framework
- SQL Server: Database management system
- Visual Studio: Development environment
